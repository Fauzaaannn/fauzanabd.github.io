Mobile Apps Project

Flutter - World Time App

Berikut merupakan hasil belajar Flutter dan Dart pertama saya, yaitu "World Time App" dimana gambar di atas menunjukkan homescreen dari aplikasi tersebut. Aplikasi ini saya buat menggunakan Android Studio. Aplikasi ini berfungsi untuk menunjukkan waktu yang ada di beberapa negara di dunia. Homescreen ini akan merubah background-nya menjadi malam apabila waktu yang ditampilkan menunjukkan waktu malam, begitu juga sebaliknya jika menunjukkan waktu siang.

World Time App - World Time API

Aplikasi World Time App ini mengambil data dari API yang sudah ada di Google yaitu World Time API. Data-data tersebut saya tampilkan ke dalam emulator dan saya buat agar user bisa memilih lokasi negara apa yang mereka ingin lihat waktunya, seperti yang ada pada gambar di atas. Setelah user memilih negara pada halaman "Choose Location", maka akan kembali ke homescreen dengan menunjukkan waktu negara yang dipilih.
